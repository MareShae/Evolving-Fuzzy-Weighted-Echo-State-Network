# Evolving Fuzzy Weighted Echo State Network

The Fuzzy Weighted Echo State Network (FWESN) is a hybrid prediction model that improves upon the performance of Echo State Networks (ESN) by combining it with the concepts of the Tagaki-Sugeno models. It was proposed by Yao Zhao and Yingshun Li [1].

This small project FWESN into an evolving model through
    + evolving membership instead fuzzy c-mean
    + recursive least squares instead of matrix inverse
